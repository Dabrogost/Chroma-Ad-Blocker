# Security Policy

## Supported Versions

Currently, only the latest version of Chroma Ad-Blocker (the `main` branch) is actively supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within Chroma Ad-Blocker, please send an 
email to **dabrogost@gmail.com**. 

Please include:
- A brief description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact (e.g., bypass, data exposure).

We will acknowledge your report and work to address the issue as quickly as possible.
Please do not open public issues for security vulnerabilities.
